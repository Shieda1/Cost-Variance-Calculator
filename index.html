What is the cost variance?
The cost variance is the difference between the amount of cash that you plan to spend on a project and the actual amount that you end up spending in order to complete the project. Simply it is the difference between the budgeted cost of work performed and the actual cost of work performed.
How to calculate cost variance?
To calculate cost variance you should use the following formula:
cost variance = budgeted cost - actual cost
Example: Company x had a budgeted cost of $9,000 to repair some of it's machine but the actual repairs cost was $10,500, calculate the cost variance?
Using the cost variance formula the answer will be:
cost variance = budgeted cost - actual cost
cost variance = $9,000 - $10,500
cost variance = -$1,500
Why use cost variance?
The cost variance is usually used to track the progression of the project. It works as an indicator of how well the company monitor potential risks and how well it analyzes the data related to the project.
